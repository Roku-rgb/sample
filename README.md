## Requirements

- Git（建議使用最新版）
- Linux / macOS / Windows（含 WSL）皆可

## Quick Start

1. 複製專案

```bash
git clone <your-repo-url>
cd sample
```

2. 查看專案內容

```bash
ls -la
```

3. 開啟並閱讀 README

```bash
cat README.md
```

### 1) Edit Documentation

你可以直接編輯 `README.md`，補充專案資訊、功能說明或操作步驟。

### 2) Commit Your Changes

```bash
git add README.md
git commit -m "docs: update README usage guide"
git push
```

### 3) Continue Expanding

當專案新增程式碼後，建議在 README 補上：
